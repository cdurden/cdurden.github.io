Tools

Tool Description
Aleksi Aleksi streamlines the process of developing vocabulary and understanding grammar when reading a website in a foreign language.
PiCast PiCast is a tool that allows presenters to wirelessly stream a presentation screen to an HDMI display through a Raspberry Pi. Its purpose is to enable teachers to be mobile when presenting in a classroom.
dotpipeR dotpipeR is a tool that enables data processing and visualization. It implements computational pipelines in the DOT graph file format. These pipelines are transformed into R code and run using the R software for statistical computing. A web application enables users to navigate the inputs and outputs at various nodes in the pipeline and to visualize the data via an easily-extensible set of visualization methods.
